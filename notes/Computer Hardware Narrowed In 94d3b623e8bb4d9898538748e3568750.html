    • Basic logic gates build up & combine with some other pieces of electronics like diodes, to make more complex gates and give out a specified Boolean/Binary output. (That's why computers speak BINARY)
    • Logic gates make up Integrated Circuits (IC's)
      • 6 Classifications of IC's
        • SSI - Small Scale Integration
          • Up to 10 transistors or a few gates within a single package such as AND,OR,NOT gates.
        • MSI - Medium Scale Integration
          • 10→100 transistors / 10s of gates in a single package
          • perform digital operations such as adders, decoders, counters, flip-flops and multiplexers.
        • LSI - Large Scale Integration
          • 100→1,000 transistors / 100s of gates in a single package
          • perform digital operations such as I/O chips, memory, arithmetic and logic units.
        • VLSI - Very-Large Scale Integration
          • 1,000→10,000 transistors / 1000s of gates in a single package
          • perform digital operations such as processors, large memory arrays and programmable logic devices.
        • SLSI - Super-Large Scale Integration
          • 10,000→100,000 transistors in a single package
          • perform digital operations such as calculators.
        • ULSI - Ultra-Large Scale Integration
          • 1,000,000+ transistors in a single package
          • perform digital operations used in computers CPUs, GPUs, video processors,

      • Memory creation using logic gates combinations that form a "LATCH" / flip flops
        • A Latch holds on to a particular value and stays that way. (i.e. gated latch)
        • a latch holds one bit (a binary value)
          • latches are put side by side to store more bits
          • registers are a group of latches arranged in a grid (like 16 x 16) that hold a single binary number of different widths(or max capacity) like 32-bit or 64-bit
            • address of a latch/ memory address can be gotten using the column and row address in binary
            • 2 multiplexors use these column and row addresses
          • RAM sticks pack modules of these bit storage arrangements in massive nested matrices of memory cells
            • Different types of RAMs use different circuits to store individual bits
              • like using different logic gates, charge traps, capacitors etc
        • Inputting bit data into this logic combination (transistor circuit config.) is called writing
        • Getting data out is called reading
